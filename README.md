# 🌍 HopeConnect – Rescue Platform

HopeConnect is a web-based platform that allows users to report injured animals or homeless people and enables NGOs to respond efficiently.

---

## 🚀 Features

### 👤 User

* Register & Login
* Submit complaints with image & location
* Track complaint status

### 🏥 NGO

* View assigned complaints
* Update status (Assigned → Resolved)
* Upload "After" image as proof

### 🛠️ Admin

* View all complaints
* Assign NGOs
* Monitor complaint statistics
* Track NGO performance

---

## 📍 Key Highlights

* 📸 Image-based complaint system (Before & After)
* 📌 Location tracking (Google Maps integration)
* 📊 Admin analytics dashboard
* 🔄 Real-time complaint status updates
* 👥 Multi-role authentication system

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, Bootstrap, JavaScript
* **Backend:** Flask (Python)
* **Database:** SQLite

---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Shravanighule20/HopeConnect.git
```

2. Navigate to project folder:

```
cd hopeconnect
```

3. Run the application:

```
python app.py
```

4. Open in browser:

```
http://127.0.0.1:5000/
```
